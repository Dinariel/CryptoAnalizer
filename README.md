Программа шифрует текст, который находится тут: texts/text.txt.
Результат выводится в файл: texts/resultEncoder.txt.
Расшифрвка обратно возможна двумя способами:
1. По известному ключу (результат выводится в файл texts/resultDecoder.txt)
2. Методом BrutForce (результат будет тут: texts/resultBrutForce.txt)
